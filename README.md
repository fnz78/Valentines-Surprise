# ❤️ Romantic Heart Sealer


A beautiful, interactive web experience designed to express love and create a shared moment. This application guides the user through a journey of memories, leading to a personalized heart-drawing ritual to "seal" their love.

## ✨ Features


- **Immersive Journey**:
  - **Landing Page**: A welcoming entrance with a countdown to Valentine's Day.
  - **Memory Lane**: A visual journey through cherished memories with heartbeat sound effects and haptic feedback.
  - **Scratch Reveal**: A fun, interactive scratch card to reveal a secret message.


- **Interactive Heart Drawing**:
  - **Trace the Heart**: A guided canvas where users draw a heart to "seal" the message.
  - **Fluid Drawing**: Smooth, glowing lines with a neon effect for a magical feel.
  - **Reset Capability**: Easily reset the drawing to get it perfect.

- **Sharing**:
  - **Seal with Love**: Captures the user's unique drawing.
  - **Share**: Share the sealed heart image directly with loved ones via native sharing or download.

- **Atmosphere**:
  - **Visuals**: Floating hearts, star constellations, and gentle animations.
  - **Audio**: Subtle heartbeat sounds synchronized with visual ripples (no annoying background music).
  - **Animations**: Powered by Framer Motion for smooth, organic transitions.

## 🛠️ Tech Stack

- **React**: UI library for building the interface.
- **TypeScript**: For type-safe code.
- **Vite**: Fast build tool and development server.
- **Tailwind CSS**: Utility-first CSS framework for styling.
- **Framer Motion**: For complex animations and gestures.
- **Lucide React**: Beautiful, consistent icons.
- **Canvas API**: For the interactive scratch card and drawing features.

## 🚀 Getting Started

1.  **Install dependencies**:
    ```bash
    npm install
    ```

2.  **Run the development server**:
    ```bash
    npm run dev
    ```

3.  **Build for production**:
    ```bash
    npm run build
    ```

## 📱 Mobile Optimized

The experience is fully responsive and optimized for touch devices, ensuring a perfect experience on phones and tablets.
